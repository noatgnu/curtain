Command Help

Type commands in the chat box starting with !. Use @ to reference genes/proteins with autocomplete.
Search Commands
!searchgene Search

Search for proteins by gene names

Example: !searchgene @TP53 @BRCA1
!searchpid Search

Search for proteins by primary IDs

Example: !searchpid @P04637 @P38398
Annotation Commands
!anngene Annotate

Add/remove annotations by gene names

Add: !anngene -a @TP53 @BRCA1 Remove: !anngene -r @TP53 @BRCA1
!annpid Annotate

Add/remove annotations by primary IDs

Add: !annpid -a @P04637 Remove: !annpid -r @P04637
State Management
!savestate State

Save current visualization settings

Save: !savestate
!savestate -l State

Load a saved state by ID

Example: !savestate -l 0
!savestate -a State

List all available saved states

Example: !savestate -a
!savestate -r State

Remove a saved state by ID

Example: !savestate -r 0
!savestate -ra State

Remove all saved states

Example: !savestate -ra
!savestate -p State

Push state to other connected users

Example: !savestate -p

Selection Commands

!select Selection

Quick selection operations (alias: !sel)

Flags: !select -sig (select significant) !select -up (select upregulated) !select -down (select downregulated) !select -clear (clear all) !select -invert (invert selection)
!filter Selection

Filter and select by criteria (alias: !f)

Examples: !filter -fc 1 2 (FC between 1 and 2) !filter -p 0.05 (p-value ≤ 0.05)
!count Selection

Count proteins matching criteria

Examples: !count -sig (count significant) !count -selected (count selected) !count -all (count total)

System Commands

!help System

Show help panel or get help for a specific command

Examples: !help !help select
!rd System

Redraw all plots and refresh visualizations

Example: !rd
!clear System

Clear all chat messages

Example: !clear
!instructor System

Toggle instructor mode for teaching

Example: !instructor

Quick Aliases

!sp → !searchpid
!ag → !anngene
!ap → !annpid
!ss → !savestate
!sel → !select
