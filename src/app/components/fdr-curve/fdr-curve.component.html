About FDR Curves

Define custom False Discovery Rate thresholds across different fold change values. This allows for more nuanced significance cutoffs than a single global FDR threshold.

Note: Enabling this will override standard cutoff annotations on the volcano plot.
Format: First row should be column headers (x and y). Subsequent rows contain fold change (x) and FDR threshold (y) values separated by tabs.
